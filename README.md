# voxel-viewer-dAPP

- A deer voxel art is deployed via a single HTML page.
- Built the voxel art NFT dApp based on Scaffold ETH

```
+------+     +------+     +--------+     +--------------------------+
| .vox |---->| html |---->|contract|---->|dAPP based on Scaffold ETH|----> NFT
+------+     +------+     +--------+     +--------------------------+
```
